# 🎬 React Movie App

A modern movie browsing web application built using React and Vite.

## 🚀 Features
- View popular movies
- Search movies
- Add to favorites
- Responsive UI
- API integration using TMDB

## 🛠 Tech Stack
- React
- Vite
- JavaScript
- CSS
- TMDB API

## 🔐 Environment Variables
Create a .env file in the root folder and add:

VITE_API_KEY=YOUR_API_KEY

## ▶️ Run Locally
npm install
npm run dev

# 🎬 React Movie App

A modern movie browsing web application built using React and Vite.

## 🚀 Features
- View popular movies
- Search movies
- Add to favorites
- Responsive UI
- API integration using TMDB

## 🛠 Tech Stack
- React
- Vite
- JavaScript
- CSS
- TMDB API
